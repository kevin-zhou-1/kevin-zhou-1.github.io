Conclusion

Thatâ€™s a mouthful - but the important takeaway is that all of these distributions fundamentally describe the same natural phenomena!

We have an event that

  • has a uniform â€œtendencyâ€ to happen over time
  • has independent occurrences (memoryless)

This is why these distributions are so ubiquitous and generally useful.

For example, the number of particles that decay within a certain amount of time in a clump of U-235 can be modelled using a Poisson process. The waiting time between each decay event can be modelled as an exponential variable and the waiting time for xxxï»¿ particles to decay can be modelled as a gamma process.

Actuaries use the Poisson process to model claim numbers for a portfolio in a year and the Gamma process to model the claim cost. The latter may sound a little counterintuitive at first - but an intuitive explanation is that we can assume claims cost accumulate at a constant rate naturally over time, it is only when they are discovered (a claim being lodged) that the full claims cost (up to that point) is revealed. The claims cost is then reset back to zero and the accumulation process begins anew.
